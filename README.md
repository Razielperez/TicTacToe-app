# tic tac toe app

This project is an "tic tac toe" game for Android.

In the game there is an option to play against the computer or play against each other.

The game design simulates a realistic game with a notebook and pencil.

# Technologies:
- Java
- Android

# Why this project?
I chose to build this app in order to improve my skills in JAVA.

Get to know the work environment of Android.

Learn to work in Android Studio.

And take the first step to building my own app.
